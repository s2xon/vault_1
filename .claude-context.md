### Conn App Summary
Conn is an in-person/online meetup app. Core concepts:
- **Groups** organized by **genres** (fitness, music, tech, art, gaming, food, etc.)
- **Posts** with types: updates, accomplishments, questions, event recaps
- **Stars** as the recognition/reaction system
- **Meetups** can be in-person, online, or hybrid
- **User profiles** show groups, top accomplishments, meetup count, stars earned
- All pages are interlinked with Obsidian backlinks

### Lifting Tracker Summary
- Push/Pull/Legs split
- Table-based workout log (exercise, sets x reps, weight, notes)
- Copy-paste template for new sessions
- PR tracking table
